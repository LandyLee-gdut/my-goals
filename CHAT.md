# CHAT.md

CHAT 系统总览文档。

详细使用说明见 AGENTS.md。

## 快速命令

```bash
# 添加记录
python3 ~/.openclaw/skills/chat-memory/scripts/add_record.py

# 搜索记录
python3 ~/.openclaw/skills/chat-memory/scripts/search.py "关键词"

# 检查提醒
python3 ~/.openclaw/skills/chat-memory/scripts/check_reminders.py

# 设置提醒
python3 ~/.openclaw/skills/chat-memory/scripts/set_reminder.py
```
